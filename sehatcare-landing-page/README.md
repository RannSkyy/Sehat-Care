# SehatCare - Premium Digital Health Landing Page & Simulator

[Indonesian Version Below]

SehatCare is a highly polished, interactive premium landing page and full-featured health simulator dashboard. Designed using the **Elegant Dark** aesthetic for corporate presentation, combined with a clean, light **Interactive Sandbox** simulating real-time patient interactions on a high-fidelity mobile device.

---

## 🌟 Key Features

1. **Elegant Dark Corporate Theme**
   - High-contrast premium typography pairing utilizing **Space Grotesk** for modern bold headings and **Inter** for readable copy.
   - Smooth gradients, fine borders (`#262626`), and deep ambient neon branding hues (`#6366f1` Indigo).

2. **Interactive Live-Sync Mobile Simulator**
   - Directly control and test patient dashboard data from the sandbox panel.
   - **Personalized Digitized Medical Card**: Edit names, adjust systolic/diastolic blood pressure live, and trigger automatic hypertension indicators with color warnings.
   - **Telehealth Video Conference**: Simulated HD video consultations with interactive doctor list, screen state changes, and audio call animations.
   - **Pill Tracker & Reminders**: Add customized new prescriptions in real-time, toggle supplement completion checklist, and track status indicators.
   - **Smart Clinic Schedule**: Simulated rontgen scans, general blood tests, and doctor appointments slots.
   - **Autoplay Slideshow**: Instantly loop through all 5 key app screens with a single tap.

3. **In-depth Bento Features Layout**
   - Elegant grid representation for medical capabilities, telehealth safety protocols, Pill Tracker, and Smart Calendar.

4. **Transparent Pricing Packages**
   - Distinctive dark cards and detailed listings for Free, Pro, and Family tiers.

5. **Integrated Validation Contact Form**
   - Front-end validation with instant receipt feedback upon submittal.

---

## 🛠️ Tech Stack & Structure

- **Framework**: React 18+ with TypeScript & Vite
- **Styling**: Tailwind CSS
- **Interactions**: State-driven reactive simulation with Framer Motion (imported from `motion/react` if needed)
- **Icons**: Lucide React

---

## 🚀 Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open on your local port (default is `http://localhost:3000`).

# SehatCare - Landing Page & Simulator Medis Premium

SehatCare adalah landing page premium interaktif yang dilengkapi dengan dasbor simulator kesehatan interaktif secara penuh. Dirancang dengan estetika **Elegant Dark** (Gelap Elegan) untuk presentasi korporat yang mewah, dikombinasikan dengan area **Interactive Sandbox** berwarna terang yang bersih untuk mensimulasikan interaksi pasien secara real-time pada kerangka perangkat ponsel tingkat tinggi.

---

## 🌟 Fitur Utama

1. **Tema Korporat Gelap Elegan (Elegant Dark)**
   - Perpaduan tipografi premium berefisiensi tinggi menggunakan **Space Grotesk** untuk tajuk modern dan **Inter** untuk keterbacaan teks utama yang bersih.
   - Siluet gradasi halus, garis pembatas tipis (`#262626`), serta nuansa neon ambient indigo yang hidup (`#6366f1`).

2. **Simulator Ponsel Interaktif Sinkronisasi-Langsung (Live-Sync)**
   - Kendalikan dan uji data dasbor pasien secara langsung dari panel pengontrol sandbox sebelah kiri.
   - **Kartu Medis Digital**: Ketik nama, atur tekanan darah sistolik & diastolik secara instan, dan picu indikator hipertensi otomatis dengan peringatan warna merah.
   - **Telehealth Video**: Simulasi konsultasi tatap muka berdefinisi tinggi (HD) dengan pilihan dokter spesialis yang dinamis.
   - **Pelacak Obat (Pill Tracker)**: Tambahkan resep obat baru Anda sendiri, tandai daftar periksa selesai/tidaknya obat harian, dan pantau kemajuan kesembuhan.
   - **Kalender Jadwal Cerdas**: Daftar janji temu rontgen, tes darah umum, hingga jadwal kunjungan dokter.
   - **Slideshow Putar Otomatis**: Menjalankan transisi ke-5 layar aplikasi secara otomatis hanya dengan satu sentuhan tombol.

3. **Tampilan Grid Fitur Bento (Bento Features)**
   - Grid modern untuk menyorot keunggulan faskes terakreditasi, privasi data standar HIPAA, dan enkripsi cloud SSL medis.

4. **Paket Harga Transparan**
   - Kartu paket Free, Pro (Rekomendasi), dan Family dengan detail keuntungan lengkap pendukung hidup sehat Anda.

5. **Formulir Konsultasi Tervalidasi**
   - Dilengkapi validasi berkas formulir secara real-time disertai umpan balik penerimaan konsultasi instan yang ramah.

---

## 🛠️ Teknologi & Arsitektur

- **Framework**: React 18+ dengan TypeScript & Vite
- **Styling**: Tailwind CSS
- **Interaksi & Animasi**: Reactive state-driven simulation & Lucide Icons
